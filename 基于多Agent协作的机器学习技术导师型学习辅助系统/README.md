# 基于多 Agent 协作的机器学习技术导师型学习辅助系统

这是一个最小可运行 demo，优先验证课程设计的核心闭环：

- 学习规划 Agent
- 概念导师 Agent
- 算法可视化
- CSV 分类实验
- 结果解释
- 错误诊断
- 浏览器端学习档案 localStorage + IndexedDB

当前版本不依赖第三方包，使用 Python 标准库即可运行。Agent 回答已改为调用用户配置的真实大模型 API 生成。

## 运行方式

```powershell
python app.py
```

然后打开：

```text
http://127.0.0.1:8000
```

如果本机没有把 Python 加入 PATH，可以改用你自己的 Python 解释器绝对路径运行 `app.py`。

## 当前 demo 边界

- 后端暂用标准库 `http.server`，后续可替换为 Flask 或 FastAPI。
- Agent 已支持 OpenAI-compatible Chat Completions 接口，例如 OpenAI、通义/智谱等兼容接口、本地 Ollama 的 `/v1` 接口。
- API Key、模型名和接口地址在前端“API 设置”页面录入，保存在浏览器 localStorage 中；后端只在当前请求中使用，不长期保存。
- 如果未配置 API，学习规划、概念讲解、代码陪练、算法解释、结果解释、错误诊断等生成式回答会提示先配置接口，不再返回固定模板答案。
- 机器学习实验先实现 CSV 分类任务与最近质心分类器，后续可接入 pandas、scikit-learn 支持更多模型。
- 学习档案保存在浏览器本地，不使用服务端数据库。

## API 设置说明

在页面顶部进入“API 设置”:

- 统一配置: 勾选“所有 Agent 统一使用同一个模型接口”。
- 分 Agent 配置: 取消勾选后，可分别配置学习规划、概念导师、算法可视化、代码陪练、结果解释、错误诊断 Agent。
- 快捷复制: 填好统一接口后点击“复制到全部 Agent”，再按需要微调单个 Agent。

API 地址填写规则:

- OpenAI: `https://api.openai.com/v1`
- OpenAI-compatible 服务: `https://你的服务地址/v1`
- 完整接口地址也可以: `https://你的服务地址/v1/chat/completions`
- 本地 Ollama: `http://127.0.0.1:11434/v1`

## 学习规划交互

学习规划 Agent 会生成可点击的阶段路线。每个阶段包含学习目标、具体步骤、实践任务、完成标准和推荐跳转页面。

- 点击左侧阶段卡片: 查看该阶段详细学习路线。
- 点击“进入...”按钮: 跳转到概念学习、算法可视化、代码陪练、实验中心、错误诊断或学习档案页面。
- 点击“标记完成”: 更新当前阶段和学习进度。

## 概念学习交互

概念学习页已升级为学习工作台:

- 支持常用主题快捷选择、自定义知识点、学习者水平、学习场景和本次学习目标。
- 模型输出会展示一句话总结、通俗解释、类比、技术定义、关键点、公式/判断规则、项目流程联系、代码示例、常见误区、小练习、自测题和下一步主题。
- 下一步主题可直接点击继续学习，实践按钮可跳转到代码陪练、实验中心等页面。

## 算法可视化交互

算法可视化页已升级为多算法实验室:

- 支持 K-means 聚类、KNN 分类、线性回归和决策树分类四类算法的 Canvas 分步演示。
- 可选择数据形态，并按算法调整关键参数，例如聚类数 K、KNN 邻居数 k、线性回归学习率和决策树最大深度。
- 每一步都会同步展示观察重点、图例、核心指标和当前状态，让用户知道应该看画布上的哪些变化。
- 可视化 Agent 会结合当前画布状态生成步骤解释、参数含义、学习要点、常见误区、下一步观察和相关概念。
- 解释结果可跳转到概念学习、代码陪练或实验中心，形成“看图理解 -> 写代码复现 -> 实验验证”的闭环。

## 代码陪练交互

代码陪练页已升级为三栏式代码工作台:

- 左侧选择项目、学习者水平、代码风格、学习目标和当前机器学习流程步骤。
- 中间展示模型生成的当前步骤代码、步骤目标、预期输出和动手练习，并支持一键复制代码。
- 右侧展示为什么这样写、逐行解释、关键概念、运行检查清单、常见错误、下一步建议和相关跳转。
- 关键概念可点击进入概念学习页，相关操作可跳转到实验中心、学习档案等页面，生成结果也可以保存到本地学习档案。

## 实验中心交互

实验中心页已升级为分类实验工作台:

- 支持加载示例 CSV、上传本地 CSV、粘贴 CSV、清空数据和填写数据集名称。
- 数据训练前会自动展示样本行数、字段数量、数值特征、缺失值、重复行、类别分布和训练就绪检查。
- 字段分析区可以点击列名快捷设置目标列，减少用户手动输入列名出错。
- 训练后会展示 Accuracy、Precision、Recall、F1、混淆矩阵、各类别指标、样本预测对照和结果解释 Agent 的复盘建议。
- 支持复制可复现实验代码、导出 Markdown 实验报告、保存实验记录，并可把实验结果发送到错误诊断页继续分析。

## 错误诊断交互

错误诊断页已升级为诊断工作台:

- 支持快速填入缺失值、维度不匹配、目标列缺失等常见机器学习报错示例。
- 用户可以补充问题场景、发生阶段、严重程度、期望行为、运行环境、报错信息和相关代码。
- 诊断前会检查输入完整度，诊断后展示错误类型、根因判断、立即修复、修复代码、排查步骤、验证步骤、原理解释和预防清单。
- 诊断结果可复制修复代码、导出 Markdown 诊断报告、保存到本地学习档案，并能跳转到概念学习或代码陪练继续学习。

## 学习档案交互

学习档案页已升级为可点开的本地档案工作台:

- 顶部展示总记录、学习路线、实验/诊断记录和最近更新时间，便于快速判断学习沉淀情况。
- 左侧支持关键词检索、按记录类型筛选、按时间或类型排序，并可点击概览项快速过滤。
- 中间历史列表展示每条记录的类型、时间、标题、摘要和标签，点击任意记录后会在右侧展开完整字段。
- 详情区支持查看原始 JSON 字段、复制 JSON，并可根据记录来源跳转到学习规划、概念学习、代码陪练、实验中心或错误诊断继续学习。
